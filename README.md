# HepsiVar PWA

Mobil uyumlu, Vercel'de statik olarak calisan PWA.

## Ozellikler

- Gorev, alisveris, not ve harcama ekranlari
- LocalStorage ile cihazda veri saklama
- Service worker ile cevrimdisi acilis
- Manifest ile ana ekrana eklenebilir PWA

## Vercel

Vercel'de framework preset olarak `Other` secilebilir. Build komutu gerekmez, output dizini repo kokudur.
